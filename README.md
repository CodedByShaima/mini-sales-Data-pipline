# Mini Sales Data Pipeline

## Overview
A mini data pipeline that processes and analyzes retail sales data.

## Tools Used
- Python
- Pandas
- SQLite
- Matplotlib

## What I Did
- Loaded and explored a real sales dataset (9,800+ records)
- Cleaned and transformed the data
- Extracted monthly sales trends
- Identified top 5 best-selling products
- Stored clean data in a SQLite database
- Visualized insights using charts

## Key Insights
- November consistently has the highest sales across all years
- Canon imageCLASS 2200 is the top-selling product ($61,599)
